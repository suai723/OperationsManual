## 去重规则

每次写入前，对目标二级分类目录下的已有条目文件进行相似度判断，满足以下任一条件即视为重复：

| 判断标准 | 说明 |
|---------|------|
| 标题高度相似 | 新条目标题与已有条目标题编辑距离≤2，或语义完全一致 |
| 核心内容重叠 | 新内容的主要信息点（≥70%）已被已有条目覆盖 |
| 来源相同且内容无更新 | 相同来源文档的相同章节已存在对应条目 |

**检测到重复时的处理方式：**

- **单条模式**：暂停写入，提示用户：
  ```
  ⚠️ 检测到疑似重复条目
  已有条目：<entry_ID> - <标题>（路径：<路径>）
  请选择：
  [1] 跳过，不写入
  [2] 覆盖并创建新版本（原文件保留，新文件标注 replaces）
  [3] 强制作为新条目写入（并行保存）
  ```
- **批量模式（子 Agent 执行时）**：自动跳过重复条目，记录到回报文件的 `skipped` 列表中，由主 Agent 在最终结果中统一展示
